The connectors in `composeApp/src/commonMain/kotlin/de/selfmade4u/tucanpluskmp/connector` implement html parsing of pages from TUCaN.
Example html pages are stored in `composeApp/src/commonTest/resources`.

For example the test in `composeApp/src/commonTest/kotlin/de/selfmade4u/tucanpluskmp/GeneratedExamResultsTest.kt` exercises this parsing.
